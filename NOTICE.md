# Data attribution

## World Cities Database

`input/worldcities.csv` and the city-index artifacts derived from it are based
on the [SimpleMaps World Cities Database](https://simplemaps.com/data/world-cities).

The SimpleMaps Basic World Cities Database is licensed under the
[Creative Commons Attribution 4.0 International License](https://creativecommons.org/licenses/by/4.0/).
The database is provided by [SimpleMaps.com](https://simplemaps.com/).

The source snapshot bundled in this repository predates this project's 1.0.0
release. Its original download date was not recorded. Generated artifacts are
deterministic derivatives of that checked-in snapshot.
